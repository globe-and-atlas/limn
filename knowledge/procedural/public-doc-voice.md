# Public Documentation Voice

## 2026-05-24 — Separate Public Science From Claim Notes

- Public-facing Limn science docs should explain mechanisms, utility, prior art, and original project contributions in neutral third-person language.
- Avoid second-person coaching language such as "you can claim," "you own," or "what you own" in public docs and in-app science panels.
- Keep direct ownership and publication-strategy notes in `LIMN_CLAIM_NOTES.md`.
- Public docs may attribute original Limn composites to Daniel Bally / Globe & Atlas, but should not read like private IP coaching.
